irc-bot
====

This is my node.js implementation of an IRC bot.  



Job Information comes back like the following:

    {
      "name":"JobName",
      "url":"JobUrl",
      "build":
      {
        "number":1,
        "phase":"STARTED",
        "status":"FAILED",
        "url":"job/project/5",
        "full_url":"http://ci.jenkins.org/job/project/5"
        "parameters":
        {
          "branch":"master"
        }
      }
    }
